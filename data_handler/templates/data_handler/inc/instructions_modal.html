Instructions File Upload
NOTE:

The number of records that will be used to predict the likelihood of donors donating depends on the quality and format of the file provided. To ensure data validations and correct values formatting, please follow these instructions before uploading your donor file.

What Do You Need?

It is prudent to note that your donor file MUST contain the following column (information)

  1. Donor Unique Identifier (ID)
  2. At least three (3) types of data points (text columns) from the list below.
    1. NOTE: In order for the model to run, please supply a minimum of three (3) text columns on your donor file.
      1. Donation history is NOT a part of this requirement.
    2. Recommended: Country, State, City and Zip Code level data to see Donor Predicted via Geo Location (map) feature.

List of Useful Data (Information)

Your donor file should contain the following columns/information (highly recommended). Use unique headers for each column. Each column should have a different header name. This helps the data validation correctly classify the information in the column.

  • Donation History (Example: 2019 Donation, 2018 Donation, 2017 Donation)
  • Donor Age
  • Gender
  • Marital Status
  • Occupation
  • Education Level
  • Income
  • Past Engagement (Example: Volunteer, Mentor, Board Member)
  • Past Giving Areas (Health, Education, Program-related etc.)
  • Email Address
  • Phone Number
  • Geo Location for Map feature
    • Country
    • State
    • City
    • Zip Code
NOTE:

Your donor file must be a .csv,.xls, or .xlsx. Your donor file may also contain any number of additional columns. All data points will be returned with the prediction(s) appended file in the CSV format. Each data column should contain the same type of information. For example, if you have a column with the header “State,” then you should only include States in that column. Also, please eliminate all unusual characters from the file, if possible (for example, $, %, &, commas, etc.).
